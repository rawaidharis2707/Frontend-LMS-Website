# 🎓 Lumina International Academy - Learning Management System (LMS)

![Lumina Academy Banner](https://images.unsplash.com/photo-1523050335456-c38a7046D397?auto=format&fit=crop&q=80&w=1200&fm=webp)

## 📌 Overview
**Lumina International Academy** is a state-of-the-art Learning Management System designed to bridge the gap between students, teachers, and administrators. Built with a focus on modern aesthetics (SF Pro design language) and performance, this platform provides a unified digital ecosystem for academic excellence, administrative efficiency, and seamless communication.

## 🚀 Key Features

### 👨‍🎓 Student Portal
- **Dashboard:** At-a-glance view of attendance, subjects, and upcoming tasks.
- **Academics:** Access to lecture notes, assignments, and quizzes.
- **Finance:** View and print fee vouchers, check payment history and fines.
- **Timetable:** Interactive schedule for classes and exams.

### 👩‍🏫 Teacher Portal
- **Management:** Track student attendance and upload marks/remarks.
- **Resources:** Upload lectures, assignment descriptions, and study notes.
- **Dynamic Access:** Delegates administrative tasks (e.g., timetable input) based on permissions.
- **Attendance:** Monitor personal attendance records.

### 🏢 Admin Portal
- **Admissions:** Streamlined processing for new student applications.
- **Academics:** Manage class distribution, subject assignments, and timetable inputs.
- **Finance:** Complete discount management, fee voucher generation, and fine tracking.
- **HR:** Manage teacher attendance and performance records.

### 🔐 Super Admin Portal
- **System Logs:** Comprehensive activity logging for security auditing.
- **HR & Finance:** Role distribution, salary management, and global financial overviews.
- **Configurations:** High-level system settings and reporting tools.

## 🛠️ Technology Stack
- **Frontend:** HTML5, CSS3 (Vanilla CSS with Custom Properties)
- **Framework:** [Bootstrap 5.3.2](https://getbootstrap.com/)
- **Icons:** [Font Awesome 6.4.2](https://fontawesome.com/)
- **Logic:** Vanilla JavaScript (ES6+)
- **Storage:** LocalStorage-based data persistence (for demo functionality)
- **Assets:** High-quality optimized images from Unsplash.

## 🎨 Design System
Lumina uses a **Premium Royal Blue & Sky Blue** color palette, emphasizing clarity and readability:
- **Primary:** `#1E3A8A` (Deep Royal)
- **Accent:** `#0EA5E9` (Sky Blue)
- **Typography:** SF Pro Display (System Font Stack for native feel)
- **Components:** Glassmorphism effects, smooth micro-animations, and responsive layouts.
